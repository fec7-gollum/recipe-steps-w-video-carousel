## Usage

 This app contians 10,000,000 unique pages. Example: http://localhost:3030/10

## Requirements

- Node 12
- mysql

## Installing Dependencies

From within the root directory:

```sh
npm install
```

## Build Bundle

```sh
npm run build
```

## Start Server

```sh
npm run start
```

## Seed Database

```sh
npm run seed:db
```
